# README Generator

I will help you create a comprehensive `README.md` for your project.

## Structure

1.  **Project Title & Description**: Clear and concise.
2.  **Tech Stack**: List frameworks and libraries used.
3.  **Getting Started**:
    - Prerequisites
    - Installation commands
    - Running the dev server
4.  **Features**: Bulleted list of key capabilities.
5.  **Project Structure**: Tree view of key directories.
6.  **Contributing**: How to help.
7.  **License**: License information.

## Steps
1.  Analyze the `package.json` to understand dependencies.
2.  Look at the file structure to generate the project tree.
3.  Draft the README content based on the template above.
4.  Write to `README.md`.
